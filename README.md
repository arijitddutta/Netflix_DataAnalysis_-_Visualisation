# Netflix_DataAnalysis_&_Visualisation
This project explores the Netflix Movies and TV Shows dataset to uncover patterns in content type, release trends, genres, ratings, and countries of production. Using Python (Pandas, NumPy) for data cleaning and analysis, and Matplotlib/Seaborn for visualization, the project aims to answer questions such as: How has Netflix's content library grown over time? What genres and content types dominate the platform? Which countries produce the most content? And how are ratings distributed across the catalog? The goal is to turn raw tabular data into clear, visual insights that reveal trends in Netflix's content strategy over the years.
